API 중 exception 이라는 예외처리들


예외처리는 보험처리와 같은 느낌이다.
예외가 발생할 수도 있는 부분을 try로 감싸준다.
try {
	System.out.println("i / j = " + (i / j));  // 예외가 발생 할수도 있고, 안할수도 있는 부분에 예외처리를 한다.
}catch(ArithmeticException e){ 
System.out.println(e.getMessage());  //예외 메세지 출력.
}	

이부분에서 ArithmeticException 을 Exception 으로 써도 무방할지도 모른다.
